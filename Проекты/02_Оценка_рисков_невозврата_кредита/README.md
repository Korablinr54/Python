# Тренировочный проект №2: Оценка рисков невозврата кредита

## Описание
Проект по анализу кредитных рисков с помощью Pandas. Цель: изучить факторы невозврата кредитов (debt), предобработать данные заемщиков и проверить гипотезы о влиянии семьи, дохода, возраста. Данные: 21 525 записей о клиентах банка.

## Цели проекта

### Предобработка данных
**Цель**: Очистить данные для корректного анализа.
- Обнаружение и обработка пропусков (total_income, days_employed)
- Устранение дубликатов и неявных дубликатов (education, gender)
- Выявление и удаление выбросов (total_income IQR, dob_years=0, children>5)
- Подсчет потери данных (~10-15% строк)

### Категоризация клиентов
**Цель**: Сегментировать заемщиков для группового анализа.
- Доход: 5 категорий (очень низкий → высокий) по квантилям
- Возраст: до 40 / после 40 лет
- Дети: без детей / 1-2 / 3+
- Образование: стандартизация (primary, secondary, etc.)

### Исследовательский анализ
**Цель**: Найти корреляции с просрочками (debt=1).
- Влияние дохода, возраста, детей, образования на долю должников (pivot_table + barplot)
- Проверка гипотез:
  1. Клиенты с детьми ответственнее? (с детьми выше риск)
  2. Одинокие мужчины низкого дохода vs семейные среднего (первые чаще должники)
- Доп. исследование: цели кредита (недвижимость > образование > авто)
