# Тренировочный проект: Python и Pandas для анализа данных доставки

## Описание
Проект для изучения Python и Pandas через анализ данных доставки заказов. Цели: освоить базовый синтаксис, работу с DataFrame и написание функций для обработки таблиц. Данные — CSV с заказами доставки.

## Цели проекта

### Базовый Python
**Цель**: Освоить фундаментальные конструкции языка.
- Решение задач с циклами, условиями, функциями
- Обработка строк и списков (замены, сортировка, конвертация типов)
- Валидация пользовательского ввода
- Парсинг адресов с помощью регулярных выражений

### Работа с Pandas
**Цель**: Научиться анализировать табличные данные.
- Создание и манипуляция Series/DataFrame
- Загрузка и очистка реального CSV (переименование колонок)
- Разбиение большой таблицы на логические подтаблицы
- Фильтрация данных с помощью .query() (по продуктам, районам, транспорту)
- Обновление значений в таблице по условиям

### Функции для DataFrame
**Цель**: Практика написания переиспользуемых функций.
- Автоматическое приведение названий колонок к snake_case
- Обработка смешанных форматов колонок ("First Name" → "first_name")
