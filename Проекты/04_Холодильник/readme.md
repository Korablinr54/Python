# Учет продуктов со сроком годности

Python-проект для управления складом продуктов с контролем срока годности. Реализует базовый функционал для учета пищевых товаров.

## Основные функции

### `add(items, title, amount, expiration_date=None)`
Добавляет продукт в систему с указанием названия, количества и опционально - срока годности.

### `add_by_note(items, note)`
Добавляет продукт из текстовой строки формата "Название количество [дата]". Автоматически парсит компоненты.

### `find(items, needle)`
Ищет продукты по названию с использованием регистронезависимого поиска.

### `amount(items, needle)`
Рассчитывает общее количество продуктов, найденных по заданному поисковому запросу.

### `expire(items, in_advance_days=0)`
Определяет продукты с истекающим или истекшим сроком годности. Позволяет настраивать период предупреждения.

## Технические особенности
- Использование `Decimal` для точных расчетов количества
- Работа с датами через модуль `datetime`
- Словарная структура хранения данных в памяти
- Поддержка частичного поиска и фильтрации

## Назначение проекта
Демонстрация навыков работы с Python, структурами данных, обработкой дат и реализацией бизнес-логики для управления инвентарем.
